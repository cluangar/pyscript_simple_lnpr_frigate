Use on Home Assistant to send event to LNPR API

# Prerequisites
1. Addons pyscript
2. LNPR API: docker pull cluangar/simple_lnpr_frigate

# Running
1. Add pyscript to Hass
2. Add Automation yaml to Hass
3. Install LNPR API Docker
